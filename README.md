# AnyDesk_Reset
This batch script 🖥️ automates the process of generating a new AnyDesk ID 🔄 on Windows systems. It disables the AnyDesk service 🛑, removes related settings from ProgramData and local user directories 🗑️, and restarts the service 🔄. Compatible with both 32-bit and 64-bit systems 🧑‍💻, ensuring a fresh AnyDesk configuration.
